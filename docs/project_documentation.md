# Honeypot ELK Attack Monitoring

This project implements a honeypot-based cyber attack monitoring system using Cowrie and the ELK Stack.

## Components

Cowrie Honeypot – captures attacker interactions.

Filebeat – forwards logs.

Elasticsearch – stores logs.

Kibana – visualizes attacks.

## Features

Real-time attack monitoring  
Visualization of attacker IP addresses  
Command tracking  
Login attempt monitoring
